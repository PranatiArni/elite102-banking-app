# elite102-banking-app

Banking App Elite 102 README
 
 A simple command-line banking system that allows users to make a variety of actions on bank accounts, built in Python and mySQL


 Features:

- Create a new bank account with an initial deposit
- Deposit money into the account
- Withdraw money from the account
- Check the account balance
- View transactions
- List all the accounts
- Delete an account
- Menu-driven CLI terminal 
- Unit tests with 'unittest' python library

Technologies Used:

- Python
- MySQL
- msql-connector-python
- VS Code
- Github

Setup Intstructions:

1. Clone repository

2. Install Dependancies

3. Set Up MySQL Database
- open my SQL Workbench or VS code DB Notebook
- Create database

4. Configure Database Creation in main.py with mySQL credentials

5. cd into your project

6. Run the application: python3 main.py

7. Run Unit Tests
